Configure Fluid Properties (PVT)

Set fluid properties for liquid loading diagnostics

Why is this needed?

PVT properties are used to calculate critical gas velocity (Turner criteria) for liquid loading diagnosis. Accurate values improve the reliability of the LIQUID_LOADING_FLAG.

  • API Gravity: Higher API = lighter oil (typical: 25-45)
  • Gas Specific Gravity: Ratio to air (typical: 0.6-0.8)
  • Water Salinity: Affects water density (typical: 10,000-50,000 ppm)
  • Temperature: Reservoir temperature in °F
  • Tubing ID: Inside diameter for velocity calculations
