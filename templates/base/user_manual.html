1. Introduction

Welcome to the Gas Lift Opportunity Automation System – a smart web tool that helps you find which wells are good candidates for gas lift installation.

What is Gas Lift?

Gas lift is a method used in oil production to help wells produce more oil. Think of it like adding a straw to a drink – it helps push the oil to the surface when the well's natural pressure isn't strong enough anymore.

Why This System?

In many oilfields, engineers manually review hundreds of wells to decide which ones need gas lift. This is time consuming and can be inconsistent. This system automates that process – it analyses well test data, identifies wells with declining trends, and ranks them so you know exactly which wells to focus on.

Step 2: Map Your Columns

After uploading, the system shows the Column Mapping page. This is where you tell the system which column in your file corresponds to each required parameter.

  • For each required parameter, you'll see a dropdown menu.
  • Select the column name from your file that matches.
  • The system will try to auto match columns based on common names – check that it's correct.
  • If a column is missing, the system will highlight it in red.

Once columns are mapped, you can preview your uploaded data to make sure everything looks right.

5. Understanding the Behind the Scenes Logic

The system uses two statistical tools:

  • Mann Kendall Trend Test – checks if there's a real, consistent upward or downward pattern.
  • Sen's Slope Estimator – calculates how fast the change is happening (mild, moderate, or aggressive).

How the Ranking Works:

  1. The system calculates a "trend score" = (trend direction) × (magnitude) for each parameter.
  2. It multiplies each score by your weight for that parameter.
  3. It adds all scores together to get a total Candidate Score.
  4. Higher score = higher rank (1, 2, 3...).

6. Troubleshooting Guide

Upload failed – File format not supported
Make sure your file is .xlsx or .csv format.
Column mapping error – Required column not found
Check that your file has all 10 required columns and map them correctly.
No wells found – data is empty
Ensure your data has more than just a header row.
Analysis runs but no wells are ranked
Your wells may be stable with no significant trends, or your data window is too short.
Date format error
The system expects standard date formats like YYYY-MM-DD.
The system is slow
If you have more than 10,000 rows, consider reducing your dataset.

7. Frequently Asked Questions (FAQ)

Do I need to be a statistician to use this?
Absolutely not. The system does all the math for you.
Can I trust the system's recommendations?
Yes – the system follows the same logic that experienced engineers use.
Can I analyze multiple fields at once?
Yes, as long as your data file includes all wells from different fields.
How often should I run the analysis?
We recommend running it whenever you have new well test data (e.g., monthly, quarterly).
Is my data safe?
Yes – all data is encrypted and stored securely.

9. Next Steps After Using the System

  1. Review the top 5-10 wells – these are your highest priority.
  2. Double check with field data – visit the well site if possible.
  3. Build a detailed well model for the top candidates to confirm gas lift benefits.
  4. Create a workover/intervention plan – schedule the gas lift installation.
  5. Monitor results – after gas lift is installed, run the analysis again to see improvement.
